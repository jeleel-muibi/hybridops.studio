# CI/CD Pipelines

CI/CD pipelines for HybridOps.Studio: image builds, infrastructure provisioning, configuration, DR drills, and platform verification. Each entry documents **what** the pipeline proves and **where** the evidence lands.

> Access flags (`public`, `academy`, `mixed`) follow the documentation access model in [ADR-0021](../adr/ADR-0021-docs-access-model.md).

## Pipeline tiers

- **core** — required for normal platform operation (image builds, base infra, drift checks).  
- **extended** — recommended pipelines (DR drills, burst tests, HPC extensions).  
- **experimental** — spikes or prototypes that may change or be retired.
